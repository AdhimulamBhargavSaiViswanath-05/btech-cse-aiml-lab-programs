# Information Retrieval Systems Lab

## Lab Name
Information Retrieval Systems Lab

## Semester
III-II (6th Semester)

## Short Description
This folder contains information retrieval experiments including text preprocessing, indexing, vectorization, classification/clustering artifacts, and notebook-based implementations.

## List of Programs / Experiments
- Core notebooks: `Exp1.ipynb` to `Exp5.ipynb`, `EXP1-_11.ipynb` to `EXP4_11.ipynb`
- Additional notebooks: `26082025submission.ipynb`, `29082025.ipynb`, `todays4.ipynb`, `with exp4 all things.ipynb`
- Python scripts: `roch.py`, `vocab.py`, `vocab_builder.py`
- Experiment modules:
  - `Newly_Exp6/` (Flask-style app + templates/static)
  - `NewlyExp9/` (social media sentiment analysis assets)
  - `NewlyExp10/` (citation analysis dataset/assets)
  - `newsexp6/webscrap/` (news scraping notebook and output)
- Data/model resources:
  - `input_files2/`, `newsgroups_full_texts/`
  - `indexes/`, `exp4_models/`, `exp5_models/`

## Technologies Used
- Python
- Jupyter Notebook
- JSON/Text corpus processing
- Information retrieval and text mining workflows

## Key Concepts Covered
- Tokenization, stemming, and vocabulary building
- Inverted indexing and metadata mapping
- Vector-space style representations
- Classification and clustering model artifact usage

## Folder Structure Explanation
- Root notebooks/scripts: main experiment implementations
- `input_files2/`, `newsgroups_full_texts/`: document corpora
- `indexes/`: generated indexing outputs
- `exp4_models/`, `exp5_models/`: saved model/vectorizer artifacts
- `Newly_Exp6`, `NewlyExp9`, `NewlyExp10`: focused advanced experiments
