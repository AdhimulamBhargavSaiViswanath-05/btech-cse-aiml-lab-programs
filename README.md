# B.Tech CSE (AI & ML) Lab Programs

This repository is a curated, semester-wise portfolio of B.Tech CSE (AI & ML) laboratory work, including notebooks, source code, datasets, and experiment outputs across core subjects.

## Repository Scope

- Academic lab programs organized by subject and semester progression
- Practical implementations in Python, R, JavaScript, HTML/CSS, and Node.js
- Supporting outputs (plots, simulations, and model artifacts) where required for reproducibility

## Lab Modules

- **computer-networks-lab** – socket programming, networking fundamentals, and NS2/animation outputs
- **deep-learning-lab** – foundational and applied deep learning experiments
- **full-stack-development** – web basics and full-stack/Node.js practicals
- **information-retrieval-systems-lab** – indexing, retrieval, text processing, and IR experiments
- **r-programming** – statistical computing and visualization experiments in R
