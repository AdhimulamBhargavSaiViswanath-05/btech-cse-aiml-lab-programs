# Full Stack Development – Part 2 (Backend & Full Stack)

## Lab Name
Backend and Full Stack Lab (Part 2)

## Semester
III-II (6th Semester)

## Short Description
This folder includes Node.js and Express-based practical programs, experiment scripts, and mini web app components for full-stack lab activities.

## List of Programs / Experiments
- Core experiment scripts: `EXP1A.js`, `EXP1B.js`, `EXP1C.js`, `EXP2B.js`, `EXP11A.js` to `EXP11E.js`, `2a.js`, `2b.js`, `3a.js`, `3b.js`, `11epractical.js`
- Utility/demo scripts: `chat.js`, `file.js`, `exp4a.js`
- `exp4b/`: Express server mini project (`server.js`, `index.html`, package files)
- `exp15b/`: multi-page static web module with stylesheet
- `Node JS Programs/`: auxiliary Node.js practice files

## Technologies Used
- JavaScript (Node.js runtime)
- Express framework
- HTML/CSS for UI pages
- npm package management

## Key Concepts Covered
- Node.js scripting and module usage
- HTTP server creation with Express
- Basic route handling and app structure
- Integration of frontend pages with backend flow

## Folder Structure Explanation
- Root `.js` files: experiment-wise backend scripts
- `exp4b/`: package-managed Node/Express exercise
- `exp15b/`: static multi-page UI module
- `Node JS Programs/`: supplementary practice programs
