# Full Stack Development Lab

## Lab Name
Full Stack Development Lab

## Semester
III-II (6th Semester)

## Short Description
This folder contains full stack development practicals split into web fundamentals and backend/full-stack implementations.

## List of Programs / Experiments
- `part-1_WebBasics/`: HTML, CSS, JavaScript foundational experiments (`EXP 1` to `EXP 9`)
- `part-2_full-stack/`: Node.js, Express, and practical backend/full-stack exercises

## Technologies Used
- HTML5, CSS3, JavaScript
- Node.js
- Express
- MongoDB ecosystem references (in project dependencies)

## Key Concepts Covered
- Page structure and semantic HTML
- Styling and layout with CSS
- JavaScript DOM and object usage
- Backend server setup and routing basics

## Folder Structure Explanation
- `part-1_WebBasics/`: Frontend/web basics lab tasks
