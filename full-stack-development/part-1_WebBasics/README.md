# Full Stack Development – Part 1 (Web Basics)

## Lab Name
Web Basics Lab (Part 1)

## Semester
III-II (6th Semester)

## Short Description
This folder contains foundational web development experiments focusing on HTML structure, CSS styling, and introductory JavaScript.

## List of Programs / Experiments
- `EXP 1`: Links, lists, profiles, image gallery basics
- `EXP 2`: Tables, frames, forms, and related HTML pages
- `EXP 3`: Embedded media and CSS introduction
- `EXP 4`: Basic webpage with stylesheet
- `EXP 5`: CSS properties (color, box model, fonts, background)
- `EXP 6`: JavaScript input/output and validation basics
- `EXP 7`: JavaScript built-in objects
- `EXP 8`: Conditional logic and simple problem programs
- `EXP 9`: Form validation and function-driven mini tasks

## Technologies Used
- HTML
- CSS
- JavaScript

## Key Concepts Covered
- HTML elements and document layout
- Form handling and user input
- CSS styling and box model
- JavaScript control flow and browser objects

## Folder Structure Explanation
- `EXP 1` to `EXP 9`: Experiment-wise folders containing program files for each practical topic
